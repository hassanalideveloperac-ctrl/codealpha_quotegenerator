# 💬 Quote Generator App

## 📋 Overview
A beautiful and inspiring quote generator application built with Flutter for the CodeAlpha Internship program. Discover random quotes from famous personalities, save your favorites, and revisit your most inspiring moments.

## ✨ Features

### 📝 Quote Display
- **Random Quotes**: Get a new inspiring quote with each tap
- **Author Attribution**: Every quote shows the original author
- **Elegant Design**: Beautiful gradient cards with quote marks
- **One-Tap Refresh**: Generate new quotes instantly

### ❤️ Favorites Management
- **Save Favorites**: Tap heart icon to save quotes you love
- **Favorites Screen**: View all saved quotes in one place
- **Remove Favorites**: Easily remove quotes from favorites
- **Clear All**: Remove all favorite quotes at once

### 📜 Recent History
- **Auto-Save**: Your last 20 viewed quotes are saved automatically
- **History View**: Toggle to see your recent quotes
- **No Duplicates**: Each quote appears only once in history

### 🎨 User Interface
- **Gradient Backgrounds**: Stunning purple gradients
- **Smooth Animations**: Fade and scale transitions
- **Responsive Design**: Works on all screen sizes
- **Material Design 3**: Modern Flutter design

## 🛠️ Technologies Used

| Technology | Purpose |
|------------|---------|
| Flutter | Frontend Framework |
| Provider | State Management |
| SharedPreferences | Local Data Storage |
| Material Design 3 | UI Components |
| Flutter Launcher Icons | Custom App Icon |

## 📲 Installation

### Prerequisites
- Flutter SDK (3.0.0 or higher)
- Android Studio / VS Code
- Android Emulator / Physical Device / Chrome Browser
